Anthropic ⇄ OpenAI gateway · Go

A Go gateway that lets Claude Code talk to ~20 third-party LLM providers

Claude Code speaks the Anthropic Messages API. Most upstream providers speak OpenAI chat-completions. This gateway sits in between, translating and routing every request — a clean-room Go reimplementation of musistudio/claude-code-router (Node.js, MIT), byte-compatible with its ~/.claude-code-router/config.json.

Byte-compatible config

Same capitalised Providers/Router JSON shape as the Node original — an existing install, including claude_toolkit-managed ones, needs no changes.

No Node.js runtime

A single Go binary, ccr (built from cmd/ccr), instead of a node/npm dependency.

Modern transport

Gin, optional HTTP/3 over QUIC (TLS-gated), and brotli-first content negotiation, all covered by table-driven tests.

Honest streaming & errors

SSE events flushed individually; upstream status codes preserved and re-shaped into the Anthropic error envelope, never collapsed to a generic 502.
